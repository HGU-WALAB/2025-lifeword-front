## 📝 소개
'라이프워드'는 교회의 설교와 성경 말씀을 효율적으로 관리하고 공유할 수 있는 웹 서비스입니다. 목회자들은 설교를 작성하고 관리할 수 있으며, 성도들은 설교를 열람하고 성경을 쉽게 찾아볼 수 있습니다.

### 주요 기능
- 설교 작성 및 관리
- 설교 버전 관리 시스템
- 성경 구절 빠른 검색
- 설교 북마크 기능
- 관리자 페이지를 통한 사용자/설교 관리
- 소셜 로그인 (카카오, 구글)
- 자체 로그인 시스템
